# Conventions: test-feature-check

## Naming conventions
- Files: kebab-case
- Functions: camelCase
- Types: PascalCase
- Constants: UPPER_SNAKE_CASE

## Architecture
- Plugin architecture with hooks and tools
- State machine pattern for workflow phases

## Testing
- Unit tests for pure functions
- Integration tests for state transitions

## Code style
- TypeScript strict mode
- Early returns for error cases
- Log with log.info/warn/error
